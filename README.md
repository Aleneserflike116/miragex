## 🛠 What this tool does

This tool locks your files so only you can open them. It works on digital documents, photos, and archives. It uses AES-256-GCM for encryption. This is the same standard banks use to protect transactions. It also uses Argon2id to turn your password into a strong digital key. This combination keeps your data safe from intruders. 

You can use miragex for:
* Encrypting tax documents.
* Protecting personal identity records.
* Securing private journals or notes.
* Hiding sensitive work files.

## 🔑 How to encrypt a file

Encryption changes your file into a jumbled format that no one can read without your password.

1. Open the miragex application from your desktop or start menu.
2. Select the "Encrypt" tab.
3. Click the "Browse" button to choose the file you want to lock.
4. Type a strong password in the box. Use a mix of letters, numbers, and symbols.
5. Click "Start" to begin the process.
6. The software will create a new file with an extra tag to show it is encrypted. You can now delete or store the original file as you wish.

## 🔓 How to decrypt a file

Decryption reverses the process, turning your file back into its original form.

1. Run the miragex application.
2. Select the "Decrypt" tab.
3. Click "Browse" and select the encrypted file you want to open.
4. Type the same password you used during the encryption step.
5. Click "Start."
6. The software will save the unlocked file in the same folder as the encrypted version.

## 🛡 Security considerations

The security of your files relies on your password. If you lose your password, the software cannot recover your data. The tool works offline, so it does not keep a backup of your password anywhere. Store your password in a safe place, like a physical notebook or a dedicated password manager. Never share your password with others. Because this tool functions entirely on your computer, your data never leaves your hardware. This prevents interception of your sensitive files by third parties.
